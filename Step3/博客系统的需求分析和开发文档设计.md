## 文章列表

路由：GET /v1/articles
参数：

    query
    {
        page: 1,
        size: 10
    }
    
    
返回：

    {
        code: 0,
        message: "获取列表成功",
        data: {
            articles: [
                {

        }
    }

## 文章详情

路由：GET /v1/articles/:articleId
    
参数：

    params
    {
        articleId: "文章id"
    }
    
返回：

    {
        code: 0,
        message: "获取列表成功",
        data: {
            article: {
                title: "文章标题",
                content: "文章内容"
                createdAt: timestamp
            }
        }
    }

    }
    
返回：

    {

    }
